The hero reel is controlled by reel-config.json. Follow these steps:

01
Drop videos into /videos/ folder
Supported: .mp4, .webm — Compress under 15MB each via HandBrake (handbrake.fr)
02
Edit reel-config.json to list your files
Open in any text editor. Add filenames to the "clips" array. Change "interval_seconds" to control timing.
03
Drag folder onto Netlify
netlify.com → Your site → Deploys → drag eons-site/ folder. Live in ~60 seconds.
💡 Pro tip
Compress renders in HandBrake — H.264, 1920×1080, RF 23–26. Keeps the site fast on mobile.
